Simple web app to keep track of your debts.

## Frontend

```
cd frontend
```

### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### Run your tests
```
npm run test
```

#### Lints and fixes files
```
npm run lint
```

## Backend

```
cd backend
```
